# 사주 조회 — 만세력 (萬歲曆)

한국천문연구원(KASI) 기반 만세력(1900~2050) 데이터로 사주팔자를 계산하고 해석하는 **자체완결형 웹앱**입니다. 데이터가 HTML 안에 내장되어 있어 별도 서버 없이 파일 하나로 동작합니다.

## 🔗 공개 데모

**https://ksoh777-droid.github.io/ManSeaYuk1/**

> 브라우저에서 바로 이용할 수 있습니다. (AI 사주 풀이 기능은 개인 API 크레딧을 사용하므로 공개 버전에서는 비활성화되어 있으며, 로컬 실행 시에만 동작합니다. 아래 참고.)

## ✨ 주요 기능

- **사주 4기둥(년·월·일·시주)** — 십성·지장간 표시
- **만세력 역조회** — 날짜 ↔ 간지(60갑자) 상호 변환, 생일 찾기
- **진태양시 보정** — 출생지 경도·균시차·과거 표준시(127.5°E)·서머타임 자동 반영, 야자시/정자시 토글, 시각 모름 옵션
- **신살·십이운성·십이신살** — 길/흉/중 분류 (천덕·월덕귀인, 백호대살, 괴강살, 원진살 등 세분화)
- **격국·용신** — 격국(월지 정기/투출), 용신(억부법 + 조후)
- **오행/십성 막대차트 + 신강지수 게이지**
- **납음오행·명궁**
- **합충형파해** — 원국 4기둥 간 천간·지지 상호작용 자동 감지 (SVG 관계도)
- **대운·세운·월운·일진** — 연/월 선택 달력, 날짜 클릭 시 상세 해석
- **인쇄 / 이미지 저장** (html2canvas)
- **🤖 AI 사주 풀이** — Claude(Opus)가 명식을 종합해 맞춤 풀이 작성 *(로컬 실행 전용)*

## 🚀 로컬에서 실행하기

정적 파일이므로 브라우저로 `index.html`을 직접 열어도 대부분의 기능이 동작하지만, **AI 사주 풀이**를 사용하려면 로컬 서버(`server.mjs`)가 필요합니다.

```bash
# 의존성 설치
npm install

# 로컬 서버 실행 (http://localhost:8123)
node server.mjs
```

AI 풀이를 사용하려면 다음 중 하나의 인증이 설정되어 있어야 합니다.

- 환경변수 `ANTHROPIC_API_KEY` (Anthropic 콘솔 API 키), 또는
- Claude 로그인 (`ant auth login`)

인증이 없으면 계산·해석 기능은 모두 정상 동작하고 AI 풀이만 안내 메시지를 표시합니다.

## 🛠 빌드

앱 화면 소스는 `saju.template.html`이며, 여기에 만세력 데이터(`web_data.json`)와 `html2canvas`를 주입해 배포용 파일을 생성합니다.

```bash
node build_html.mjs
# → 사주.html + index.html (동일 내용, index.html은 GitHub Pages 공개용) 생성
```
